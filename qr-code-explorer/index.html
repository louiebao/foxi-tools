QR Anatomy
Finder Patterns Three corner squares that help scanners detect and orient the code at any angle
Timing Patterns Alternating stripes between finder patterns that establish the module grid spacing
Alignment Patterns Small squares (version 2+) that help scanners correct for distortion or skew
Format Information Strips bordering each finder pattern that encode the error correction level and mask pattern used
Data Area Remaining modules storing the encoded data and Reed–Solomon error correction codewords
