SQL Quine

A query that produces itself — watch it construct in real time

A quine is a program whose output is its own source code. This SELECT query is self-replicating: when you run it, SQLite returns the query text itself, character for character. The animation shows how a REPLACE() chain builds that property step by step, substituting a placeholder with the template until the output converges with the query.
